# Introduction

## Dubhe Architecture?

- [Schema](/dubhe/schemas) provides abstractions specifically designed for storing and retrieving data from a on-chain permanent storage and provide [ORM](https://en.wikipedia.org/wiki/Object-relational_mapping) capabilities across the Move ecosystem.
- [Cli](https://github.com/0xobelisk/dubhe/tree/main/packages) using for building and developing a Dubhe project.
- [Client](https://github.com/0xobelisk/dubhe/tree/main/packages) connect to the Dubhe project.
- [Create-Dubhe](https://github.com/0xobelisk/dubhe/packages/create-dubhe) quick start a Dubhe template project.
